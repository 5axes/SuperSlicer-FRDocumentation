## Fichier HTTPS CA 

### Description

Un fichier de certificat CA personnalisé peut être spécifié pour les connexions HTTPS OctoPrint, au format crt/pem.  S'il est laissé vide, le référentiel de certificats CA de l'OS par défaut est utilisé.
