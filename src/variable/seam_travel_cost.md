##  Coût déplacement

### Description

Coût du déplacement de l'extrudeuse. La pénalité la plus élevée est lorsque le point est le  plus éloigné de la position de l'extrudeuse avant l'extrusion du périmètre externe.

* Valeur par défaut : 100 %
