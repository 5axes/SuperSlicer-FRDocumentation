## Vitesse d'impression maximale

### Description

Cette fonction expérimentale est utilisée pour régler la plus haute vitesse que vous souhaitez autoriser.
Lorsque vous réglez les autres vitesses à 0, SuperSlicer calculera automatiquement la vitesse optimale de façon à garder une pression constante dans l'extrudeuse.
