* Technologie : FDM

## Annulation du décalage en Z

### Description

Définit la nouvelle valeur de lift-z pour cette dérogation.

> 0 désactivera le lift-z. 
> -1 pour désactiver. 

Peut ne fonctionner que sur les modificateurs par plage de hauteur.
