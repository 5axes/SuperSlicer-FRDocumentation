# loglevel

* Technologie : Variable générique
* Groupe : Option console
* Sous groupe : -
* Mode : -

## Logging level

### Description

Définit la sensibilité de la création du fichier log : 
- 0:fatal, 
- 1:error, 
- 2:warning,
- 3:info, 
- 4:debug, 
- 5:trace   

For example. loglevel=2 logs fatal, error and warning level messages.
